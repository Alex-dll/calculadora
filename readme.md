## How to run this project
Clone this repo
```bash
git clone https://github.com/Alex-dll/calculadora
```
Go to project directory
```bash
cd calculadora
```
Install the dependencies
```bash
npm install | yarn install
```
Start the application
```bash
npm run dev | yarn dev
```
